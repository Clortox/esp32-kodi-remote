# ESP32-ir-wifi-remote

This will be a remote for kodi and my sound systems on one device, featuring mechnaical key switches and a small display

## Completion Criteria

1) Can send commands to the kodi rest api server
2) Can send display basic metadata about the playing media on the display
3) Can auto discover the kodi server
4) Can send IR commands
5) Can send IR commands on key press
6) Custom PCB with mechanical key switch buttons
